CSS 文本属性

标签： CSS CSS基础

快速复习

在讨论CSS文本属性之前，让我们一起来回顾一些基础知识。

  1. 水平缩进属性text-indent，用于控制首行缩进。（可继承）
  2. 水平对齐属性text-align，用于控制文本行对齐方式。（可继承）
  3. 行高line-height，用于控制垂直对齐。（可继承）
  4. 用于控制垂直对齐的属性vertical-align。（不可继承）
  5. 用于控制字间距的word-spacing属性。（可继承）
  6. 用于控制字母间间距的letter-spacing属性。（可继承）
  7. 用于控制文本大小写的text-transform属性。（可继承）
  8. 文本修饰属性text-decoration。（不可继承）
  9. 文本阴影text-shadow。（不可继承）
